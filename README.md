# Career Progression and Promotion Gap Analysis for Retention Optimization at Palo Alto Networks

## Overview

This project analyzes employee career progression patterns, promotion gaps, role stagnation, and retention opportunities using machine learning and clustering techniques.

## Objectives

* Identify promotion-stalled employees
* Analyze career progression trends
* Calculate Promotion Gap Scores
* Generate Retention Opportunity Index
* Support proactive employee retention strategies

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn
* Google Colab
* Streamlit

## Machine Learning Methodology

### Feature Engineering

* Promotion Gap Ratio
* Role Stagnation Index
* Training Intensity Score
* Manager Stability Indicator

### Clustering

* K-Means Clustering
* Career Path Analysis

### KPI Metrics

* Career Cluster
* Promotion Gap Score
* Retention Opportunity Index
* Manager Stability Impact

## Project Outputs

* Career trajectory clusters
* Promotion gap risk analysis
* Retention opportunity identification
* Career intelligence dashboard
